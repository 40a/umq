umq-push(1) -- tcp/udp push message transport utility
=================================

## SYNOPSIS

`umq-push <host> <port>` [-hvV]

## OPTIONS

  `-v, --verbose`           show verbose output
  `-h, --help`              display this message
  `-V, --version`           output version

## USAGE

```
$ echo "ping" | umq push localhost 3000
```
